## substring分割字符串

这里使用的是<string.h>里面的strncp函数，具体使用示例：
```c
char buff[1024];
char * outstr = buff;
int size = fend-fstart+1;
strncpy(outstr,istr,size); 
(*outstr)[size] = '\0';
```

* 上面有两个知识点：
    * 指针必须要保证自己的地址是可以访问并且修改的，如果outstr没有指向buff这一可用内存，那么outstr可能是0x000这种不可修改内存，那么strncpy将无法访问
    * strncpy无法自动给outstr输入结束符'\0',所以需要手动输入
